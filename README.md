# Analog homebrew tap

Homebrew formula for [Analog](https://getanalog.io), a CLI that turns
webpages into structured data.

## Install

```sh
brew install getanalog/tap/analog
```

Use the full name. There's an unrelated `analog` in homebrew-core (a
logfile analyzer), and plain `brew install analog` gets you that one.
The fully-qualified form also keeps Homebrew's tap-trust scoped to
this one formula, so no separate `brew trust` step is needed.

First run:

```sh
analog login
analog browser install   # one-time ~150 MB, needed for JS-heavy pages
```

Upgrades are the usual `brew upgrade getanalog/tap/analog`.

Prefer pip? The same CLI ships on PyPI: `uv tool install analog-sdk`.

## Maintenance notes

`Formula/analog.rb` is generated by `scripts/regenerate.py`: a venv on
brew's python with every dependency as a hash-pinned wheel. pip runs
with `--no-index --no-deps`, so nothing gets resolved or fetched at
install time beyond what brew already checksummed, and installs take
seconds. Don't edit the pins by hand; CI diffs the formula against the
generator's output and will fail the PR.

To ship a new version: run the "Bump formula" workflow after the PyPI
release. A daily poller opens the bump PR if that gets forgotten.
